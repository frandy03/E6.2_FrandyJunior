Que es sass?

Es un lenguaje de hojas de estilos que se utiliza para describir el estilo de una pagina de forma limpia y estructurada. Este permite importar hojas de estilos css y su principal ventaja es que te permite usar variables.

Dispone de dos formatos diferentes para la sintaxis, lo que hace se traduce en dos extensiones de fichero diferentes: .sass y .scss.

El sass no hace uso de llaves ni punto y coma final, en busca de simplicidad y eliminación de ruido.
Ejemplo:
body
background: #000
font-size: 62.5%

Mientras que el scss permite utilizar llaves e incorporar código de CSS clásico. Ejemplo:
body {
background: #000;
font-size: 62.5%; }
