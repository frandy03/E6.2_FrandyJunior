Estructura de un elemento en HTML

  • Content

    • Es lo que contiene elemento, puede ser texto o imagen.

  • Padding

    • Es el espacio existente entre el Content y el Border.

  • Border

    • Es la línea que encierra completamente el Content y el Padding.

  • Margin

    • Es la separación opcional que existe entre la caja y el resto de cajas.

Propiedades CSS

  • Width

    • Esta propiedad se utiliza para establecer el ancho de un elemento.
    Ejemplo: { width: 100px;}

  • Max-width

    • Se utiliza para establecer el limite del ancho de un elemento.
    Ejemplo: {max-width: 500px;}

  • Height

    • Se utiliza para establecer la altura de un elemento.
    Ejemplo: {height: 200px;}

  • Max-height

    • Se utiliza para establecer el limite de la altura de un elemento.
    Ejemplo: {max-height: 600px;}

  • Display

    • Se utiliza para establecer el tipo de caja generada por un elemento.
    Ejemplo: {display: flex;}

  • Overflow

    • Se utiliza para hacer visibles o invisibles los elementos.
    Ejemplo: {overflow: hidden;}

  • Z-index

    • Esta propiedad se utiliza para definir el nivel en que se mostrara el elemento.
    Ejemplo: {z-index: auto;}

  • Line-height

    • Se utiliza para definir un valor de espacio entre una linea a otra.
    Ejemplo: {line-height: 80%;}

  • Box-sizing

    • Se utiliza para modificar las propiedades por defecto del CSS box model.
    Ejemplo: {box-sizing: content-box;}
